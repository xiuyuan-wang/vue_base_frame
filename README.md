
vue_base_frame_vite 是一套集成vue3全家桶与element-plus,通过vite3执行后台管理系统。

- Get the project code

```javascript
git clone https://github.com/xiuyuan-wang/vue_base_frame.git
```

- Installation dependencies

```javascript
cd vue_base_frame_vite
npm install
```

- run

```javascript
npm run dev
```

- build

```javascript
npm run build
```
